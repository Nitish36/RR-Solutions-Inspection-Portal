# RR Solutions Safety Inspection Portal

A full-stack industrial asset management system designed for safety compliance and certificate tracking.

## 🚀 Features
- **Secure Multi-User Auth**: Private data isolation for clients and administrative control.
- **Cloud Database**: Powered by Neon (PostgreSQL) for 100% data persistence.
- **Digital Vault**: Automated PDF certificate storage via Google Drive API.
- **Mobile Optimized**: Responsive design with QR code field verification.
- **Enterprise Analytics**: Real-time sync to Google Sheets and Looker Studio.

## 🛠️ Tech Stack
- **Backend**: Python (Flask), SQLAlchemy, Flask-Login
- **Database**: PostgreSQL (Neon.tech)
- **Files**: Google Drive API
- **Analytics**: Looker Studio / Google Sheets API
- **Frontend**: Vanilla JS, CSS3 (Responsive), Chart.js
